部署说明
支持 Docker Compose 单机一键部署（CE/EE）与 K8s 生产多副本部署两种形态
K8s 模式下 backend / portal 生产环境至少 2 副本，readiness/liveness 探针保活
CE/EE 通过 NODESKCLAW_EDITION 环境变量或 ee/ 目录存在性判断，FeatureGate 运行时控制
K8s 操作必须显式指定 --context，禁止依赖 current-context 默认值
